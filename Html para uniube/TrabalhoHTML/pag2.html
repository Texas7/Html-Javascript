Malwares

Spyware

Malware Projetado para rastrear e espionar o usuário. O spyware frequentemente inclui rastreadores de atividade, coleta de toque de tela e captura de dados. Ele também modifica as configurações de segurança do sistema.

Adware

O software com suporte a anúncio é projetado para fornecer anúncios automaticamente. Alguns tipos de adware são projetados para oferecer somente anúncios, mas é comum um adware vir acompanhado com um spyware.

Bot

Bot é o malware projetado para executar automaticamente a ação, geralmetne on-line. Enquanto a maioria dos bots é inofensivo, uma utilização progressiva de bots mal-intencionados são os botnets.

Ransomware

É um malware projetado para manter preso um sistema de computador, ou os dados incluídos nele, até que o pagamente seja feito para o sistema ser liberado. Ele funciona normalmente criptografando os dados, em outros casos ele bloqueia o sistema.

Scareware

É um tipo de malware projetado para persuadir o usuário a executar uma ação específica com base no medo. Ele simula janelas pop-up que se assemelham às janelas do SO. Na realidade nenhum problema foi detectado, a não ser que você seja induzido pelas janelas e execute o que ela pede.

Rootkit

É um malware projetado para modificar o sistema operacional e criar um backdoor. Os invasores usam o backdoor para acessar o computador remotamente. Sendo que a maioria dos Rootkits utiliza as vulnerabilidades para escalar privilégios.

Vírus

É um código executável mal-intencionado que é anexado a outros arquivos executáveis, muitas vezes programas legítimos. A maioria dos vírus necessita da ativação do usuário final. Eles podem também se espalhar pelas unidades.

Cavalo de Troia

Um cavalo de troia é um malware que realiza operações mal-intencionadas sob o pretexto de uma operação desejada. O código malicioso explora os privilégios dos usuário que o executa. Muitas vezes ele vem acompanhado de outros malwares.

Worms

São códigos maliciosos que se replicam ao explorar, de forma independente, vulnerabilidades em redes. Normalmente eles deixam a rede mais lenta. Enquanto os vírus pecisam serem executados, os worms conseguem fazer isso de forma autonoma.

MitM

Permite que o invasor tenha o controle sobre um dispositivo sem o conhecimento do usuário. Com isso o invasdor pode interceptar e capturar informações do usuário antes de tramsmiti-las ao seu destino desejado.

MitMo

Uma variação do MitM, ele assume o controle do dispositivo móvel. Quando infectado, o dispositivo é instruido a extrair as informações confidencias do usuário e envia-las para os invasores.
