Criptografia

Função Hash

MD5 (Message-Digest algorithm 5)

É uma sequência completa de operações simples binárias realizadas nos dados de entrada para produzir um resumo de mensagens de hash com 128bits.

SHA (Secure Hash Algorithm)

Funções hash criptográficas são operações matemáticas executadas em dados digitais; comparando o hash computado (a saída de execução do algoritmo) a um valor de hash conhecido e esperado, uma pessoa pode determinar a integridade dos dados. Por exemplo, calcular o hash de um arquivo baixado e comparar o resultado com um resultado hash publicado anteriormente pode mostrar se o download foi modificado ou adulterado.

Família SHA-2 (224,256,384,512): É um algoritmo mais forte e está substituindo o MD5.

SHA-256, SHA-384 e SHA-512 - Algoritmos de última geração. Quanto maior o valor em bits do SHA, mais forte será a criptografia e maior será o valor de saída, porém, mais lento será o algoritmo.

RIPEMD-160

RIPEMD-160 é uma função de hash criptográfica de 160 bits. Ele estava destinado a ser usado como uma substituição para as funções de hash MD4, MD5 e RIPEMD de 128 bits, porém, foi substituido pelo SHA-256 e SHA-512 e suas classes derivadas.
